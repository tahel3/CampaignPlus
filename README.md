# Campaign+ - מערכת לניהול קמפיינים ותרומות

 Campaign+ היא פלטפורמה מקיפה לניהול גיוס כספים, המאפשרת למנהלים ליצור ולנטר קמפיינים, ולתורמים לתמוך במטרות שונות תוך מעקב ויזואלי בזמן אמת.

---

## 🚀 תכונות עיקריות

### 🛠 ממשק מנהל (Admin)
* **ניהול קמפיינים:** יצירה, עדכון וסגירה של קמפיינים הכוללים תמונות ותיאור.
* **ניהול קטגוריות:** ארגון הקמפיינים לפי נושאים עבור סינון וחיפוש קל.
* **בקרת גישה:** מערכת הרשאות מבוססת JWT המבטיחה גישה מאובטחת למנהלים בלבד.
* **ניטור תרומות:** צפייה בהיסטוריית תרומות מפורטת לכל קמפיין בזמן אמת.

### 💳 ממשק לקוח (Client)
* **חוויה ויזואלית:** הצגת התקדמות התרומות באמצעות פסי התקדמות (Progress Bars) דינמיים.
* **עדכונים חיים:** שימוש ב-Socket.io לעדכון סכומי התרומות ללא צורך ברענון הדף.

---

## 🛠 טכנולוגיות בשימוש

* **Frontend:** JavaScript (ES6+), HTML5, CSS3, Axios.
* **Backend:** Node.js, Express.
* **Database:** MongoDB & Mongoose.
* **Real-time:** Socket.io לעדכונים מיידיים.
* **Storage:** GridFS לניהול והזרמת קבצי תמונות.



---

## 🌐 תיעוד ה-API (Endpoints)

### 🎗️ קמפיינים (Campaigns)
| פעולה | נתיב | שיטה | תיאור |
| :--- | :--- | :--- | :--- |
| שליפת קמפיינים | `/campaigns` | `GET` | מחזיר רשימה הכוללת חישוב `sumCon` לכל קמפיין. |
| יצירת קמפיין | `/campaigns` | `POST` | הוספת קמפיין חדש למערכת. |
| עדכון קמפיין | `/campaigns/:id` | `PUT` | עדכון פרטי קמפיין קיים לפי מזהה. |
| מחיקת קמפיין | `/campaigns/:id` | `DELETE` | הסרת קמפיין מהמערכת. |

### 📂 קטגוריות (Categories)
| פעולה | נתיב | שיטה | תיאור |
| :--- | :--- | :--- | :--- |
| שליפת קטגוריות | `/categories` | `GET` | מחזיר את כל הקטגוריות המוגדרות במערכת. |
| הוספת קטגוריה | `/categories` | `POST` | יצירת סיווג חדש לקמפיינים. |
| מחיקת קטגוריה | `/categories/:id` | `DELETE` | הסרת קטגוריה קיימת. |

### 💰 תרומות (Contributions)
| פעולה | נתיב | שיטה | תיאור |
| :--- | :--- | :--- | :--- |
| ביצוע תרומה | `/contributions` | `POST` | הוספת תרומה ועדכון יעד הקמפיין ב-Live. |
| תרומות לקמפיין | `/contributions/campaigns/:id` | `GET` | שליפת היסטוריית תרומות לקמפיין ספציפי. |
| סטטיסטיקות | `/contributions/stats` | `GET` | נתוני תרומות לצורך תצוגה גרפית. |

---
### 🔄 עדכונים בזמן אמת עם Socket.io
הפרויקט עושה שימוש ב-Socket.io כדי ליצור חוויית משתמש דינמית ללא צורך בריענון הדף:

צד השרת: ברגע שמתקבלת תרומה חדשה בראוט ה-POST, השרת מחשב את הסכום המעודכן ושולח הודעת donationUpdate לכל הלקוחות המחוברים.

צד הלקוח: הדפדפן מאזין לאירוע זה ומעדכן מיידית את פס ההתקדמות (Progress Bar) ואת סכום היעד המוצג על המסך.
